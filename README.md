# Помічник трейдера — Smart Trading Assistant

Україномовний ШІ-помічник для **поступового** росту капіталу.
Зовні просто, всередині — глибокий аналіз, новини, фундаментал і жорсткий
контроль ризику. Спот-торгівля на Binance.

> ⚠️ **Це навчально-дослідницький інструмент. Він не обіцяє прибутку.**
> Реальні гроші вмикаються лише після backtest-гейту, додавання API-ключів
> і явного підтвердження. За замовчуванням — Paper Trading без грошей.

---

## Швидкий старт

```bash
pip install -r requirements.txt
uvicorn api.main:app --reload      # один сервер: і інтерфейс, і API
# відкрийте у браузері:  http://localhost:8000
```

Перевірка без сервера:
```bash
python -m demo          # сценарії рішень Risk/Signal
python -m demo_full     # повний цикл + звіт
python -m pytest -q     # 40 тестів
```

Головний цикл: **Start → Watch live → Stop → Review → Improve → Repeat.**

## Як новини впливають на рішення

Це ключова зв'язка: News Engine оцінює настрій і довіру джерел, а Signal
Engine **блокує** вхід проти сильної новини. Перевірено: при сильній
негативній регуляторній новині актив отримує 0 угод (усі відхилено), тоді
як актив із нейтральними новинами торгується нормально.

## Безпека Live (три незалежні запобіжники)

Реальні гроші вмикаються ЛИШЕ якщо одночасно виконано:
1. **Backtest пройдено** — гейт вимагає profit factor > 1, додатний
   expectancy, ≥20 угод, просадку < 25%.
2. **API-ключі Binance** у `.env` (лише торгівля; вивід коштів вимкнено).
3. **Явне підтвердження** користувача галочкою в інтерфейсі.

За замовчуванням: `enabled=False`, `dry_run=True`. Emergency stop при
досягненні лімітів просадки/дня/тижня.

### Як увімкнути реальну торгівлю (коли будеш готовий)
1. Створи на Binance API-ключ: увімкни **Spot Trading**, вимкни **Withdrawals**.
2. Скопіюй `.env.example` → `.env`, встав ключі.
3. В інтерфейсі: обери Live → «Запустити backtest» → постав галочку підтвердження.
4. Кнопка старту розблокується лише коли всі умови виконані.

### Свідомі компроміси
- **Дані демо — синтетичні** (середовище розробки без інтернету). Провайдери
  `CcxtProvider` і `CryptoPanicProvider` готові — у Codex з мережею
  підставляються без зміни архітектури (див. AGENTS.md).
- **Метрики на синтетиці нереалістично високі** — на реальних даних будуть
  значно нижчі. Це чесно й очікувано.

### Залишкові ризики
- Сигнальна логіка проста — це база, а не доведена перевага. Валідувати на
  живих даних перед реальними грошима (кілька тижнів Paper).
